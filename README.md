Задание 1

Программа отображает видео с веб-камеры или из файла и позволяет отмечать точки на изображении.

При нажатии левой кнопки мыши на кадре рисуется прямоугольник в месте клика.
Клавиша C очищает все отметки, клавиша Q или кнопка интерфейса завершает работу программы.

Запуск

Камера:

python app.py camera

Видео файл:

python app.py video.mp4
Задание 2

Программа выполняет наложение одного видео на экран, находящийся в другом видео.

Сначала определяется область экрана на первом кадре, после чего программа отслеживает её положение на последующих кадрах с помощью ключевых точек и гомографии. В найденную область выполняется перспективное наложение второго видео.

Запуск
python hw2.py insert_video.mp4 camera_video.mp4

где

insert_video.mp4 — видео, которое будет вставлено

camera_video.mp4 — видео с экраном

Результат

Обработанное видео сохраняется в файл:

result.mp4

и также отображается в окне во время выполнения программы.
